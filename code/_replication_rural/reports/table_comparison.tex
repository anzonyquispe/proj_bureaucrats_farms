      header text; reorder stats; relabel Election FE $\rightarrow$ ``Electoral Cycle FE'',
      Time FE $\rightarrow$ ``Relative Time FE'', Province Trend FE $\rightarrow$
      ``Province Linear Time Trend FE''; drop Mean DV3.

\item \textbf{Section 5} (\_app\_6): header text; hyphenate ``Month-Year FE''.

\item \textbf{Section 6} (\_app\_7): hyphenate ``Month-Year FE''.

\item \textbf{Section 7} (\_app\_8): header text; remove prefoot empty row;
      relabel ``Assembly'' $\rightarrow$ `` N Assembly Constituencies '';

\item \textbf{Section 8} (\_app\_9): remove `` - Rural Grids'' from header (keep ``x'');

      so \texttt{ymean} is last.

\vspace{1em}

\noindent\textbf{Not fixable from the dofile alone:} the analysis scripts
store \texttt{ymean}/\texttt{acq} as \texttt{estadd local} (pre-formatted
strings like ``160.300''). Since \texttt{esttab}'s \texttt{fmt()} only
applies to scalars, the 3-decimal precision in Dropbox vs Overleaf's 1-
or 2-decimal display cannot be changed without editing each analysis
script's \texttt{string(mean, "\%9.3f")} $\rightarrow$ \texttt{"\%9.1f"}.
